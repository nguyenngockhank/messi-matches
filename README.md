# Messi Matches

See demo at [https://messi-matches.netlify.app/](https://messi-matches.netlify.app/)

## Script

### Fetch new matches data
```
npm run fetch
```
new matches will be store at `frontend/js/matches.js`

## Run Front end

Open `frontend/index.html` with your browser
